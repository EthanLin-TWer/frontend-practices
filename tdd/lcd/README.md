# 编程小道场

把数字转化为 lcd 灯的表现形式，打印在控制台上。下面是 0~9 的样子：

```
._.   ...   ._.   ._.   ...   ._.   ._.   ._.   ._.   ._.
|.|   ..|   ._|   ._|   |_|   |_.   |_.   ..|   |_|   |_|
|_|   ..|   |_.   ._|   ..|   ._|   |_|   ..|   |_|   ..|
```

例子: 910

```
._. ... ._.
|_| ..| |.|
..| ..| |_|
```
